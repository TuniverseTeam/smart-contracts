# WastedLands contract

### development
```
  npm install

  truffle console
```

### Compile
```
  truffle compile --all
```
### testing
```
  truffle test
```

### verify

```
  truffle run verify Tuniverse@{contract-address} --network testnet
```

### migrate
```
  truffle migrate
```

### deploy 
```
  truffle migrate --network testnet
```
### migrate specific file 
```
  truffle migrate -f 1 --to 1 --network testnet
```
